# 专业技能

## 1. 编程语言
熟练掌握 C++ 与 Python，具备面向对象设计与工程化开发能力，熟悉 C++ 新特性及 STL 标准模板库；具备基于 Flask 的后端开发经验，掌握 RESTful API 设计与接口联调；掌握 Vue 前端开发，具备前后端分离项目实践经验；熟悉 MySQL 数据库设计与优化。

## 2. 数据结构与算法
熟悉常见数据结构（数组、链表、二叉树等）及常用算法（快速排序、冒泡排序等）。

## 3. 计算机视觉
熟悉计算机视觉相关技术，熟练使用 OpenCV，了解神经网络及其在图像处理中的应用，具备基于 YOLOv5 等深度学习模型的目标检测实践经验，掌握工业相机的标定与应用开发。

## 4. 机器人开发
熟悉机器人系统调试，具备计算机视觉模块与电控系统的联调经验，掌握传感
器数据采集与融合、运动姿态解算、串口传输与运动控制及调试，熟悉 MiniPC 平台（NVIDIA Jetson、Intel NUC、树莓派、Arduino）的开发与调试，具备算法移植与优化能力。

## 5. Linux
熟悉 Linux 环境下的多线程编程与进程间通信，掌握常用的线程同步与互斥机制，熟悉使用 Linux 常用命令、Shell 脚本。

## 6. 工具链
熟练使用 Vim、GCC、GDB、Git 等工具，能够编写 Makefile 以支持大型项目编译。
